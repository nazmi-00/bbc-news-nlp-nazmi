# 📰 BBC News Article Categorizer (NLP Final Project)
```

```
An intelligent web application built with Streamlit and Scikit-Learn that
automatically classifies BBC News articles into 5 distinct categories:
**Business, Entertainment, Politics, Sport, or Tech**.
```

```
*Please open in Google Chrome for a better experience*
```

```
---
```

```
## 📰 Project Architecture
```

`1. **Dataset**: BBC News Raw Text (`bbc-text.csv`)` 

`2. **Feature Extraction**: TF-IDF (Term Frequency-Inverse Document Frequency) + N-Grams` 

`3. **Classification Models**:` 

- `**Multinomial Naive Bayes** *(Selected as Final Model due to highest accuracy)*` 

- `Support Vector Machine (SVM)`
